Nhiệm vụ:

1. Phân tích các thay đổi và thông tin mới từ các Knowledge Object nguồn.
2. Xác định chính xác những phần nào trong File mục tiêu cần được đồng bộ, cập nhật hoặc bổ sung để nhất quán với các Object nguồn.
3. Giải thích rõ:
   - Những phần cần cập nhật trong File mục tiêu.
   - Lý do cập nhật.
4. Xuất lại toàn bộ nội dung mới của File mục tiêu sau khi đã đồng bộ.

Nếu không cần cập nhật File mục tiêu, ghi rõ: No Synchronization Required

Yêu cầu:
- Chỉ cập nhật các phần thực sự bị ảnh hưởng trong File mục tiêu.
- Giữ nguyên toàn bộ các nội dung không liên quan khác của File mục tiêu.
- Không tự ý tạo thêm tri thức mới nằm ngoài dữ liệu được cung cấp.
- Xuất đầy đủ nội dung của File mục tiêu sau khi đã cập nhật để có thể ghi đè trực tiếp lên file hiện tại.
